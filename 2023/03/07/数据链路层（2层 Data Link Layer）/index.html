数据链路层（2层 Data Link Layer）

  1. 属于2层
  2. 传输单元：帧
  3. 帧结构

​ 帧头：6+6+2=14字节

​ MTU值：1500字节(国内)

​ 帧尾：4字节

​ 帧头内容：目标MAC、源MAC、类型

​ 类型的作用：识别上层协议

  1. 工作在数据链路层的设备：交换机/网卡

802.11无线网卡支持

mac地址12位六进制

  1. 交换机工作原理

    目标mac 源mac 源ip 目标ip 源端口 目标端口

收到一个数据帧后：

(1)首先学习帧中的源MAC地址来形成MAC地址表

(2)然后检查帧中的目标MAC地址，并匹配MAC地址表：如表中又匹配项，则单播转发；如表中无匹配项，则除接受端口外广播转发

(3)MAC地址表的老化时间默认是300秒(可修改)

  1. 交换机的端口：E F G Te
  2. 交换机基本工作模式及命令

第一次配置网络设备，需要使用console线，在PC需要使用“超级终端”或其他软件

1）用户模式：

switch>

可以查看交换机的基本简单信息，且不能做任何修改配置

2）特权模式：

switch> enable

switch#

可以查看所有配置，且不能修改配置，但可以做测试、保存、初始化等操作

3）全局配置模式：

switch# configure terminal

switch(config)#

默认不能查看配置！可以修改配置，且全局生效！

4）接口配置模式：

switch(config)# interface f0/1

switch(config-if)#

5）console口/线/控制台模式：

switch(config)#line console 0

默认不能查看配置！

可以修改配置，且对console口生效
